Debian10иЈ…MySQL

2022-05-03

еңЁDebian 10 зҡ„й»ҳи®ӨиҪҜд»¶жәҗдёӯе№¶дёҚеҢ…еҗ« MySQL иҪҜд»¶еҢ…пјҢе®ғе·Іиў« MariaDB жӣҝд»ЈгҖӮд»Һе®ҳзҪ‘дёӢиҪҪ MySQL иҪҜд»¶й…ҚзҪ®еҢ…пјҢй…ҚзҪ®еҗҺиҝӣиЎҢе®үиЈ…гҖӮ

е°Ҷ MySQL APT еӯҳеӮЁеә“ж·»еҠ еҲ°зі»з»ҹ

е…ҲеҲ°MySQLеӯҳеӮЁеә“дёӢиҪҪйЎөйқўпјҢдёӢиҪҪжңҖж–°зҡ„еҸ‘иЎҢеҢ…пјҢдёӢиҫ№дёӨз§ҚдёӢиҪҪж–№ејҸйҖүдёҖдёӘе°ұеҘҪгҖӮ

wget https://repo.mysql.com//mysql-apt-config_0.8.22-1_all.deb

curl https://repo.mysql.com//mysql-apt-config_0.8.22-1_all.deb

дёӢиҪҪе®ҢжҲҗеҗҺпјҢйҖҡиҝҮд»ҘдёӢе‘Ҫд»Өе®үиЈ…

sudo dpkg -i mysql-apt-config_0.8.13-1_all.deb

дҪ е°ҶдјҡзңӢеҲ°MySQLй…ҚзҪ®е®үиЈ…иҸңеҚ•пјҢйҖүжӢ©дҪ иҰҒе®үиЈ…зҡ„зүҲжң¬

й»ҳи®ӨйҖүжӢ©дәҶ MySQL 8.0пјҢеҰӮжһңиҰҒе®үиЈ… MySQL 5.7пјҢиҜ·йҖүжӢ©MySQL Server пјҶ Clusterз„¶еҗҺйҖүжӢ©еҜ№еә”зҡ„ MySQL зүҲжң¬гҖӮ
и°ғж•ҙе®ҢжҜ•йҖүйЎ№пјҢе…үж Үз§»еҠЁеҲ°OKпјҢеӣһиҪҰзЎ®е®ҡгҖӮ

дҪҝз”Ёд»ҘдёӢе‘Ҫд»Өжӣҙж–°иҪҜд»¶еҢ…пјҢе№¶е®үиЈ…MySQL

sudo apt update
sudo apt install mysql-server

е®үиЈ…иҝҮзЁӢдёӯе°Ҷдјҡеј№еҮәи®ҫзҪ® MySQL root еҜҶз Ғзҡ„еҜ№иҜқжЎҶпјҢи®ҫзҪ®з§ҳеҜҶ

жҺҘдёӢжқҘпјҢдјҡжҳҫзӨәдёҖжқЎж¶ҲжҒҜпјҢйҖҡзҹҘжӮЁжңүе…іж–°зҡ„ MySQL 8 иә«д»ҪйӘҢиҜҒзҡ„дҝЎжҒҜгҖӮ еңЁйҖүжӢ©й»ҳи®Өзҡ„ MySQL 8 иә«д»ҪйӘҢиҜҒжҸ’д»¶д№ӢеүҚпјҢиҜ·зЎ®дҝқжӮЁзҡ„еә”з”ЁзЁӢеәҸж”ҜжҢҒе®ғгҖӮ

йҖүжӢ©й»ҳи®Өз¬¬дёҖйЎ№пјҢжҢүеӣһиҪҰз»§з»ӯгҖӮ

е®үиЈ…е®ҢжҲҗеҗҺпјҢMySQL жңҚеҠЎдјҡй»ҳи®ӨиҮӘеҠЁиҝҗиЎҢпјҢеҸҜд»ҘйҖҡиҝҮд»ҘдёӢе‘Ҫд»ӨжҹҘзңӢжңҚеҠЎиҝҗиЎҢзҠ¶жҖҒ

sudo systemctl status mysql

иҝһжҺҘеҲ°MySQLжңҚеҠЎеҷЁ

еңЁз»Ҳз«ҜдёӯеҸҜд»Ҙиҫ“е…Ҙmysqlе®ўжҲ·з«Ҝе‘Ҫд»ӨжқҘиҝһжҺҘmysqlжңҚеҠЎ

mysql -u root -p

иҝңзЁӢй“ҫжҺҘMySQLж—¶иҝңзЁӢиҝһжҺҘжқғйҷҗй»ҳи®ӨжІЎжңүеҜ№еӨ–ејҖж”ҫпјҢеңЁжң¬жңәйҖҡиҝҮдёҠиҫ№зҷ»е…ҘMySQLеҗҺпјҢжӣҙж”№ вҖңmysqlвҖқ ж•°жҚ®еә“йҮҢзҡ„ вҖңuserвҖқ иЎЁйҮҢзҡ„ вҖңhostвҖқ йЎ№пјҢд»ҺвҖқlocalhostвҖқж”№з§°вҖҷ%вҖҷгҖӮ

use mysql
update user set host = '%' where user ='root'
flush privileges;

йҮҚеҗҜMySQLжңҚеҠЎеҚіеҸҜе®ҢжҲҗгҖӮ

  1. еҗҜеҠЁпјҡmysqld start
  2. еҒңжӯўпјҡmysqld stop
  3. йҮҚеҗҜпјҡmysqld restart
